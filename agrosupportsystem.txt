AgroSupportSystem

Информационно-аналитическая система поддержки принятия решений 
при выращивании овощных культур с интеграцией модуля расчета удобрений.

Есть два варианта разработки:
1 вариант (простой):
	Создать просто несколько страниц в HTML + CSS. Без серверной части вообще.
	Просто красивые ссылающиеся друг на друга страницы. 
	На JS прописать калькулятор удобрений.
Для сохранения истории запросов в калькуляторе можно использовать 
localStorage — это встроенное API браузера, которое позволяет сохранять данные на устройстве пользователя.
Вместо базы данных на сервере можно использовать:
Если данные не нужно сохранять на стороне сервера 
и достаточно хранить их только на устройстве пользователя, можно использовать локальные хранилища браузера :
	LocalStorage : Хранит данные в виде ключ-значение. Подходит для небольших объемов данных (до 5–10 МБ).
	SessionStorage : Аналогично LocalStorage, но данные удаляются после закрытия браузера.
	IndexedDB : Более мощная альтернатива для хранения больших объемов данных.
	Использование файлов JSON

Актуально: сначала сделать все максимально просто - без JSON, без ещё каких-то других приколов.
Потом можно уже добавить поиск, вывод футера и хедера с помощью JS и так далее.

Разработка основных страниц
Начните с создания базовых HTML-страниц и добавления контента.

a. Главная страница
Создайте index.html.
Добавьте:
	Заголовок и описание системы.
	Навигацию (меню) по разделам.
	Визуально привлекательный дизайн (например, баннер с изображением полей или овощей).
b. Информация о культурах
Создайте cultures.html.
Добавьте:
	Список культур (можно сделать аккордеон или карточки).
	Подробные страницы для каждой культуры (например, tomato.html, cucumber.html).
	Описание, рекомендации по выращиванию, сорта.
c. Калькулятор удобрений
Создайте calculator.html.
	Реализуйте форму с полями для ввода данных (культура, площадь, тип почвы и т.д.).
	Напишите JavaScript для расчета удобрений (например, используйте данные из fertilizers.json).
	Добавьте возможность сохранять историю запросов (например, через localStorage).
d. Информация об удобрениях
Создайте fertilizers.html.
Добавьте:
	Описание типов удобрений.
	Рекомендации по применению.
	Примеры использования.
e. О проекте
Создайте about.html.
Добавьте:
	Информацию о целях проекта.
	Контакты разработчика (форма обратной связи или ссылки на соцсети).

установка стилей фреймворков
Локальные файлы
Если вы хотите избежать зависимости от интернета, вы можете скачать файлы фреймворков и подключить их локально.

a. Для Tailwind CSS
Сгенерируйте собственный CSS-файл с помощью Play CDN или инструментов сборки.
Сохраните файл tailwind.css в папке вашего проекта (например, css/tailwind.css).
Подключите его в HTML:
html
<link rel="stylesheet" href="css/tailwind.css">
b. Для Bootstrap
Скачайте файлы Bootstrap с официального сайта .
Сохраните их в папке проекта (например, css/bootstrap.min.css и js/bootstrap.bundle.min.js).
Подключите их в HTML:
html
<link rel="stylesheet" href="css/bootstrap.min.css">
<script src="js/bootstrap.bundle.min.js"></script>
Преимущества локальных файлов:
Работает без интернета.
Полный контроль над файлами.
Недостатки:
Необходимо обновлять файлы вручную при выпуске новых версий фреймворков.
